# Review Checklist

Use this checklist before marking a content unit complete.

## Audience Fit

- [ ] The unit is written for the defined audience.
- [ ] The reader/user outcome is clear.
- [ ] Assumed prior knowledge is appropriate.

## Scope Fit

- [ ] The unit stays within scope.
- [ ] The unit does not introduce unsupported promises.
- [ ] The unit does not duplicate another unit unnecessarily.

## Content Quality

- [ ] The structure is clear.
- [ ] Examples are specific.
- [ ] Claims are sourced or framed appropriately.
- [ ] Key terms match the style guide.

## Production Readiness

- [ ] Formatting follows the style guide.
- [ ] Tables, lists, and callouts are consistent.
- [ ] Open questions are resolved or recorded.
- [ ] The next unit can build on this one.
